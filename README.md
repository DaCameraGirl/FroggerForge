# FroggerForge

A standalone browser arcade game inspired by classic road-and-river crossing mechanics.

This repo is intentionally separate from PixelForge so it can stand on its own as a small JavaScript Canvas game.

## Play

Open `index.html` in a browser.

Controls:

- Arrow keys or WASD to move
- Click or tap an adjacent lane/cell to move toward it
- R to restart

## Features

- Traffic lanes with cars and trucks
- River lanes with moving logs and turtles
- Five homes to fill per level
- Level speed increases
- Lives, score, timer, and best score
- No external game engine required
